## What I'm building: (one sentence)

I am building a beach day tracker that checks Hawaii weather and helps me decide if it is a good day to go to the beach.

## Which API I'm using: (name and URL)

I am using the Open-Meteo Weather Forecast API: https://open-meteo.com/en/docs

## Why I chose this: (what interests you about this data?)

During the summer, I often have to rely on the weather app before deciding whether to go to the beach, so I thought this would be a perfect thing to make. I chose this API because it gives real weather forecast data. I am interested in using data like temperature and UV index to decide whether beach conditions are good or not.

## Core features: (3-5 things it will do)

- Show the current weather for Hawaii beach locations.
- Display beach-related weather details like temperature, rain chance, wind speed, and UV index.
- Give a simple beach day rating, such as "Great," "Okay," or "Not ideal."

## What I don't know yet: (what about fetch/async/JSON is new to you?)

I am still practicing how to use `fetch()` to get data from an API and bring it into my project. I am also learning how `async`/`await` works because it takes time for the weather data to load. Reading nested JSON is still new to me, so I sometimes run into errors while figuring out where the data is stored.
