# Mini Project #3

## What your app does
Beach Day Tracker helps users decide if it is a good day to go to the beach on Oahu. Users can choose from saved beaches, search for a beach, refresh the weather, and see a simple beach rating based on temperature, rain chance, wind speed, and UV index.

## Which API you used (with link)
This app uses the Open-Meteo Forecast API to get current weather.

[Open-Meteo Forecast API](https://open-meteo.com/en/docs)
